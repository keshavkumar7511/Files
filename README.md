# Flange Coupling CAD Project

## Software Used
SolidWorks

## Project Description
Designed and assembled a rigid flange coupling consisting of:
- Flange
- Shaft
- Key
- Bolt
- Nut

## Skills Demonstrated
- 3D Part Modeling
- Assembly Design
- Engineering Drawings
- Interference Detection
- GD&T Basics

## Files Included
- SolidWorks Part Files (.SLDPRT)
- SolidWorks Assembly File (.SLDASM)
